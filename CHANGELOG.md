# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-01-XX

### Added
- Initial release of fmod_flutter plugin
- Support for Android and iOS platforms
- Core FMOD functionality:
  - Initialize FMOD Studio system
  - Load FMOD banks from assets
  - Play/stop events
  - Set event parameters
  - Control event volume
  - Pause/resume events
  - System update method
- Comprehensive documentation and examples
- Platform channel architecture for native integration

### Known Limitations
- Requires manual FMOD Engine setup (libraries not included)
- iOS implementation uses placeholder code (requires FMOD C API integration)
- No web support (FMOD doesn't support web)
